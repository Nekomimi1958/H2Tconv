変換の設定

以下は、変換についての詳細設定です。
なお、インデントや囲み文字列等を指定する際、タブは \t、半角スペースは \s としてください。

行

桁数
　一行の桁数を指定します。
　hr の長さや、p、center、div 等での位置決めに用いられます。

桁数で改行
　桁数で指定した幅でテキストを強制的に改行します。
　ただし、pre、xmp 内では無効となります。

連続空行制限
　連続する改行のみの行数を制限することができます。

見出し後を空けない
　見出しの後に空行が入らないようにします。

見出し

H1〜H6
　見出しの前に付ける文字列を指定します。

太字・斜体・下線

囲み
　該当部分の前後に付加する文字列を指定します。

HR罫線

罫線文字
　hr の罫線用文字列を指定します。半角でも全角でもかまいません。

箇条書き

インデント
　インデントを行うのに使う文字列です。

マーク
　ul の時のマーク文字を指定します。

数字は全角
　ol の時に付ける数字を全角にします。

表

セル区切り
　セルの区切り文字列です。

画像

代替文字列
　チェックすると、alt の内容で表示します。

デフォルト
　alt が指定されていない場合、この文字列を表示します。

ファイル名
　画像のファイル名(src="〜")を挿入します。
　入力がインターネットからの場合や、base が指定されている場合は、絶対指定に変換します。

囲み
　表示文字列の前後に付加する文字列を指定します。

リンク

挿入する
　リンク先の内容(href="〜")を該当部分の後ろに挿入します。
　入力がインターネットからの場合や、base が指定されている場合は、絶対指定に変換します。

外部リンクのみ
　外部リンク(http:〜 または mailto:〜) のみを挿入します。

前後に改行を
　リンク先の前後に改行を入れます。

囲み
　リンク先の前後に付加する文字列を指定します。

コンテンツ区切り罫線

罫線文字
　区切り罫線用の文字列を指定します。半角でも全角でもかまいません。

<div> class
　指定した class の div 要素の前に罫線を挿入します。; (セミコロン)で区切って複数指定できます。
　例:　post;navigation

<section>　 <article>　 <nav>
　HTML5で用いられる section、article、nav 要素の前に罫線を挿入します。

指定部分の削除

指定した class または id の要素を削除します。
削除対象となるのは、div、table、ul、ol、dl、section、article、nav です。

class
　削除したい部分の class を指定します。; (セミコロン)で区切って複数指定できます。

id
　削除したい部分の id を指定します。; (セミコロン)で区切って複数指定できます。
