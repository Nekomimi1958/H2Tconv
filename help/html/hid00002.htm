インストール

動作環境

Windows XP 以降なら多分だいじょうぶです。

インストールするファイル

適当なディレクトリに以下のファイルを置いてください。Program Files などOSの管理下にあるフォルダに入れると色々と面倒なことになるので、別の場所にディレクトリを作ってインストールすることをおすすめします。
バージョンアップの際は、ファイルを上書きしてください。

H2Tconv.EXE　プログラム本体
H2Tconv.CHM　このヘルプファイル

起動すると、H2Tconv.INI というファイルが H2Tconv.EXE のあるディレクトリに作られ、設定情報はこのINIファイルに保存されます。レジストリは使用していません。

なお source ディレクトリにはソースコード(C++Builder 10.2.3 用)がありますので、自由にご利用ください。


アンインストール

インストールしたファイルおよび INIファイル等を削除して下さい。
