置換の設定

ここでは、変換後のテキストに対する文字列置換を設定できます。

項目を追加するには、検索文字列 と 置換文字列 を入力して追加ボタンを押してください。
検索文字列 を / (スラッシュ)で囲むと正規表現とみなされます。入力内容に誤りがあると、赤文字表示になって追加ボタンが押せません。
置換文字列 を空欄にすると、マッチした 検索文字列 の削除になります。
上部の一覧では、空白などが分かりやすいように "検索文字列" → "置換文字列" というように引用符で囲んで表示されます。

例1:　全角空白を半角空白に置換
　"　" → " "
例2:　タブ文字を削除
　"/\t/" → ""
例3:　連続する半角空白やタブ(混在可)を1個の半角空白に置換
　"/[ \t]{2,}/" → " "


内容を変更したい場合は、一覧の項目を選んで設定を変えてから変更ボタンを押してください。
複数の項目がある場合、上から順番に置換処理が行われます。項目の位置は↑/↓ボタンで移動できます。
項目の削除は、削除ボタンかDELキーで行ってください。

一覧の項目のチェックボックスによって、置換の有効/無効を個別に切り替えられます。
